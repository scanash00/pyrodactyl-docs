## General Application Settings

| Variable | Type | Description | Default |
|----------|------|-------------|---------|
| `SKIP_SEED` | `boolean` | Skips database seeding during initialization. | `false` |
| `APP_NAME` | `string` | Name of the application. | `Pyrodactyl` |
| `APP_ENV` | `string` | Application environment (e.g., `production`, `development`). | `production` |
| `APP_DEBUG` | `boolean` | Enables debug mode for detailed error reporting. | `false` |
| `APP_URL` | `url` | Public URL for the application. | `http://localhost` |
| `APP_TIMEZONE` | `string` | Application timezone (e.g., `UTC`, `America/New_York`). | `UTC` |
| `APP_LOCALE` | `string` | Default locale for translations. | `en` |
| `APP_FALLBACK_LOCALE` | `string` | Fallback locale for translations. | `en` |
| `APP_FAKER_LOCALE` | `string` | Locale for Faker data generation. | `en_US` |
| `APP_KEY` | `string` | Application key for encryption (32 characters). | - |
| `APP_PREVIOUS_KEYS` | `string` | Comma-separated list of previous encryption keys. | - |
| `APP_MAINTENANCE_DRIVER` | `string` | Driver for maintenance mode (`file`, `cache`). | `file` |
| `APP_MAINTENANCE_STORE` | `string` | Storage for maintenance mode data. | `database` |
| `APP_REPORT_ALL_EXCEPTIONS` | `boolean` | Reports all exceptions, even handled ones. | `false` |
| `APP_ACTIVITY_PRUNE_DAYS` | `integer` | Days after which activity logs are pruned. | `90` |
| `APP_ACTIVITY_HIDE_ADMIN` | `boolean` | Hides admin activities from logs. | `false` |
| `APP_ENVIRONMENT_ONLY` | `boolean` | Enables restricted mode, ignoring database configs. | `false` |
| `APP_SERVICE_AUTHOR` | `string` | UUID or email identifying the service author. | `unknown@unknown.com` |
| `APP_CORS_ALLOWED_ORIGINS` | `string` | Comma-separated list of allowed CORS origins. | - |
| `PTERODACTYL_USE_ASSET_HASH` | `boolean` | Enables asset hashing for JS/CSS files. | `false` |
| `PTERODACTYL_SEND_INSTALL_NOTIFICATION` | `boolean` | Sends email on server install completion. | `true` |
| `PTERODACTYL_SEND_REINSTALL_NOTIFICATION` | `boolean` | Sends email on server reinstall. | `true` |
